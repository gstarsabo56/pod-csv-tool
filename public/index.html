🚀 Auto-Process Pending Rows —
Workflow:
1️⃣ VA điền Title + Image URL + Product Type vào Sheet → cột Status để trống
2️⃣ Team lead mở tool này → nhấn Check Pending → xem còn bao nhiêu rows chưa xử lý
3️⃣ Nhấn Auto-Process All → tool sẽ: pull rows → CF upload ảnh → AI rewrite title → push kết quả về Sheet
4️⃣ VA refresh Sheet → thấy SEO Title, CF URL, Status = done + timestamp
